# Estrutura de Dados em Java

Este repositório contém atividades **construídas por mim** durante a disciplina de **Estrutura de Dados**, utilizando a linguagem **Java**, no curso de Engenharia de Software.

Ao longo da disciplina, foram desenvolvidas diversas atividades práticas. Para este repositório, selecionei **uma atividade representativa de cada tipo de estrutura de dados estudada**, com o objetivo de demonstrar meu aprendizado, organização de código e compreensão dos conceitos.

---

##  Estrutura do Repositório

Cada pasta representa uma atividade escolhida, entre outras realizadas durante a disciplina:

- **arvore-binaria**  
  Implementação de uma **árvore binária**, incluindo:
  - Caminhamento in-order
  - Caminhamento pré-order
  - Caminhamento pós-order
  - Exibição dos nós folhas
  - Relação entre o nó pai (raiz) e seus filhos  

- **fila-playlist**  
  Programa em Java para gerenciamento de uma playlist de músicas, utilizando o conceito de **Fila (FIFO)**, com funcionalidades de:
  - Inserir música
  - Consultar próxima música
  - Remover música
  - Exibir músicas restantes  

- **pilha-livros**  
  Sistema para gerenciar livros a serem lidos durante as férias, utilizando o conceito de **Pilha (LIFO)**, com opções de:
  - Inserir livro
  - Consultar pilha
  - Remover livro do topo
  - Esvaziar a pilha  

---

##  Tecnologias Utilizadas

- Java
- Conceitos de Estrutura de Dados:
  - Árvore Binária
  - Fila (Queue)
  - Pilha (Stack)
