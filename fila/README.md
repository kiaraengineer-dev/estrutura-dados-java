# Fila – Playlist de Festa 

Atividade acadêmica da disciplina de Estrutura de Dados,
com o objetivo de aplicar o conceito de FILA (Queue) em Java.

A aplicação simula o gerenciamento de uma playlist de músicas
para uma festa de final de ano, onde as músicas são executadas
na ordem em que foram inseridas (FIFO – First In, First Out).

## Funcionalidades
- Inserir músicas na playlist
- Consultar a próxima música da fila
- Remover músicas da playlist
- Exibir as músicas restantes
- Encerrar o programa

## Estrutura de Dados Utilizada
- Fila (Queue)
- Implementação com -LinkedList

## Classe
- PlaylistFesta.java: contém a lógica principal da aplicação

## Tecnologias
- Java
